# Microservice_user  
Microservice User
  
## Description
  
This microservice has that main responsabilities :
* Create a user
* Update a user
* Delete a user 
* Get all users
* Get user by ID
* Get users by status  
  
This microservice also use oauth lib to authenticate a user: github.com/diegoclair/bookstore_oauth-go/oauth
