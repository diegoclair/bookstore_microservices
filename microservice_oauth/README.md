## Start application
### Permissions first:  
For <b>Unix</b> enviroment run the comand:  
* <b>```chmod +x .docker/entrypoint.sh```</b>  

For <b>Windows</b> enviroment run the comand:   
* <b>```dos2unix +x .docker/entrypoint.sh```</b>  

Now you can run:  <br>
* <b>```docker-compose up```</b>
